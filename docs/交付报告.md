## 1. 项目概述

本项目旨在构建一个通用的 AI 语音语义理解中台，支持多模型路由、知识库问答 (RAG) 和实时联网搜索。项目包含一个基于 FastAPI 的后端服务和一个基于 React 的管理后台，提供了完善的调试、评测和管理功能。

## 2. 交付清单

### 2.1. 源代码
*   **后端代码** (`/app`, `/backend`): 包含 API 服务、对话管理、RAG 引擎等核心逻辑。
*   **前端代码** (`/frontend`): 包含管理后台的 React 源码。
*   **文档** (`/docs`):
    *   `AI语音语义理解模块 - 产品需求文档.md` (PRD)
    *   `AI语音语义理解模块 - 技术设计说明书_v2.0.md` (Tech Design)
    *   `功能完成度自查报告.md`
    *   `交付报告.md` (本文档)
*   **测试脚本** (`/tests`): 包含接口自动化测试脚本 `test_all_interfaces.py`。

### 2.2. 核心功能
1.  **多意图对话**：支持指令控制、知识库问答和通用闲聊的自动路由。
2.  **流式响应**：支持打字机效果的实时流式输出。
3.  **会话管理**：支持多会话创建、重命名、删除及历史记录持久化。
4.  **知识库管理**：支持文档上传、解析、向量化索引及检索测试。
5.  **批量评测**：支持上传 Excel 用例进行自动化批量评测。
6.  **全链路日志**：记录每次对话的 Trace ID、耗时、模型路由结果等元数据。

## 3. 部署指南

### 3.1. 环境要求
*   Python 3.10+
*   Node.js 18+
*   PostgreSQL (with pgvector)
*   Redis

### 3.2. 后端部署
1.  安装依赖：
    ```bash
    pip install -r requirements.txt
    ```
2.  配置环境变量：
    复制 `.env.example` 为 `.env`，并填入数据库连接串、API Key 等信息。
3.  启动服务：
    ```bash
    uvicorn app.main:app --host 0.0.0.0 --port 8001
    ```

### 3.3. 前端部署
1.  安装依赖：
    ```bash
    cd frontend
    pnpm install
    ```
2.  构建产物：
    ```bash
    pnpm build
    ```
3.  部署：
    将 `dist` 目录下的静态文件部署到 Nginx 或其他 Web 服务器。

## 4. 接口文档摘要

详细接口定义请参考代码中的 Swagger UI (`/docs`)。

*   `POST /api/v1/auth/login`: 用户登录
*   `POST /api/v1/chat/completions`: 发送对话请求 (支持 `stream=true`)
*   `GET /api/v1/sessions`: 获取会话列表
*   `GET /api/v1/sessions/{id}/history`: 获取会话历史
*   `POST /api/v1/admin/documents/upload`: 上传知识库文档
*   `POST /api/v1/admin/eval/batch`: 提交批量评测任务
